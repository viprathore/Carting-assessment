Technical Assessment

Objective

Implement a “BuyALL” feature that demonstrates frontend, backend, and UI/UX skills while working within an existing codebase.

Requirements
BuyALL Button

Add a BuyALL button to the cart page.

The button should be enabled only when the cart contains items.

Placement and styling should be intuitive and user-friendly.

Confirmation Modal

When the BuyALL button is clicked:

Display a modal showing the total price of all cart items.

Include two actions:

Yes – confirm and proceed with the purchase

No – cancel and close the modal

Behavior

Yes

Sends purchase data to the backend.

Handles success and error states properly.

No

Closes the modal without modifying the cart.

Technical Expectations

Clean, readable, and maintainable code

Proper frontend state management

Backend validation and API handling

Smooth frontend–backend integration

Thoughtful UI/UX decisions

Timeline

This task must be completed within 72 hours after receiving the assessment.

Submission Instructions

Push your completed work to GitHub or Bitbucket

Repository name must be: Carting assessment

Share the repository link as your final submission
